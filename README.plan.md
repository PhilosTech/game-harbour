# Game Harbour

PWA platform for narrative tabletop-style games with a **professional host**. Players join on phones; host runs sessions from **Captain's Bridge** (Мастер-мостик).

## Status

Planning. [docs/PROJECT_PLAN.md](docs/PROJECT_PLAN.md)

## Stack (planned)

Next.js, PostgreSQL, Prisma, S3 storage, Auth.js, next-intl (RU/EN), PWA.

## First game

[В поисках Эльдорадо / In Search of El Dorado](docs/games/el-dorado.md) - treasure expedition in the Amazon, grounded realism.
